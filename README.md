Графический редактор на Python, Pillow и Tkinter
 Функциональность редактора:

- Масштабирование в %
- Черно-белый режим
- Розовый и голубой фильтры
- Обрезка до нужного размера (crop)
- Отражение по горизонтали
- Поворот вокруг оси
